Step 1 — Build the Shortcut (one time only)

1
Open the Shortcuts app

Tap + in the top right to create a new shortcut. Name it "Log Expense"

2
Get Latest Screenshot

Add action: Get Latest Photos → set count to 1 → enable Include Screenshots

3
Convert image to Base64

Add action: Base64 Encode → input: Photos → Line Breaks: None

4
Call Gemini API

Add action: Get Contents of URL
URL: https://generativelanguage.googleapis.com/v1beta/models/gemini-2.5-flash:generateContent?key=YOUR_GEMINI_KEY
Method: POST · Content Type: JSON

// Request body — paste as JSON: { "contents": [{ "parts": [ { "inline_data": { "mime_type": "image/jpeg", "data": "[Base64 Encoded variable]" } }, { "text": "Extract transaction from this receipt. Return ONLY JSON: {\"amount\": number, \"desc\": \"merchant name max 30 chars\", \"cat\": \"one of: food/transport/shopping/health/entertainment/utilities/education/other\", \"date\": \"YYYY-MM-DD\", \"cur\": \"3-letter currency code default MYR\", \"notes\": \"brief item summary max 40 chars\"}" } ] }] }
5
Extract the JSON response

Add action: Get Dictionary from Input → input: Contents of URL
Then: Get Value for Key → candidates
Then: Get Item from List → First Item
Then: Get Dictionary Value → content → parts → 0 → text

6
Parse values & open Family Wallet

Add action: Get Dictionary from Input on the text result
Then: Open URLs with this URL:

https://heraline.github.io/Family-Wallet/?shortcut=1&amount=[Dictionary Value: amount]&desc=[Dictionary Value: desc]&cat=[Dictionary Value: cat]&date=[Dictionary Value: date]&cur=[Dictionary Value: cur]¬es=[Dictionary Value: notes]
💡

Use URL Encode action on each dictionary value before inserting into the URL to handle special characters

✅

Once built, tap Share → Add to Home Screen to put it on your home screen as a one-tap button!

Step 2 — Using the Shortcut daily

1
Take a screenshot

After any payment — GrabPay, Touch 'n Go, bank transfer, receipt photo — press Side + Volume Up

2
Tap "Log Expense"

Tap your shortcut from the home screen or the Shortcuts widget. Gemini reads the screenshot automatically.

3
Family Wallet opens pre-filled

Amount, merchant, category, date — all filled in. Just review and tap Save!

⚡

Works with any screenshot — bank apps, e-wallets, food delivery, shopping receipts, WhatsApp payment confirmations
